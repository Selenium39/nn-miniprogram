# 丑丑头像生成器
一个有趣的微信小程序,可以随机生成各种搞笑丑萌头像,支持一键下载。

![效果演示](./images/avatar.jpg)

## 功能特点

- 头像生成
  - 随机组合面部特征
  - 丰富的表情变化
  - 多样的发型选择
  - 各种装饰元素
  - 独特的丑萌风格

- 操作体验
  - 简单的一键生成
  - 流畅的动画效果
  - 实时预览效果
  - 快捷的下载分享

- 界面设计  
  - 简洁清爽的界面
  - Canvas绘制头像
  - 合理的按钮布局
  - 友好的操作提示

- 图片处理
  - 自动保存为PNG格式
  - 支持高清图片导出
  - 便捷的文件管理
  - 支持相册保存
